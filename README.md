# Python Employee Manager

A simple employee manager written in Python.

## Features

- Add employees
- View employees
- Search employees
- Input validation
- Console application

## Technologies

- Python 3

## How to Run

```bash
python employee_manager.py
```

## Example

```text
--- EMPLOYEE MANAGER ---
1. Add employee
2. View employees
3. Search employee
4. Exit
```
